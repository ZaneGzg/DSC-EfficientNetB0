目前在番茄叶片病害识别领域，存在资源受限场景下轻量化与高精度识别难以兼顾的问题。为解决该问题，本文基于EfficientNetB0架构，提出了融合可分离扩张卷积（SDC）和通道注意力（CAN）的改进模型SC-EfficientNetB0。本文数据集来自PlantVillage 数据库，包含 10种番茄叶片病害疾病。在数据预处理阶段进行了数据平衡、增强、划分以及归一化等操作，并在后续实验训练中采用了迁移学习和混合精度训练策略。在限定推理时间2.50s的对比实验中，本文模型准确率为99.38%，参数量4.23M，FLOPs为 0.80G。在无限时准确率与损失曲线波动小、收敛快的良好状态下，既保持了高精度识别又显著降低了计算资源，实现了精度与效率的良好平衡，优于EfficientNetB0、EfficientNetV2B0、ResNet50、DenseNet121 等主流模型。随后，限时消融实验与混淆矩阵证明了SDC和CAN的协同增效性。综上，本文提出的模型兼顾了轻量化与高识别精度的需求，具备较高的部署可行性，为农业实地应用和边缘设备上的智能病害诊断监测提供了技术支持。
